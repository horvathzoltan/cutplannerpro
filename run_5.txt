# ⭐ Hogyan működne a checkbox?

A festési terv UI‑ban:

```
[ ] Cipzáros külön blokkban
```

Ha nincs bepipálva → **mostani működés**:

```
SZÍN: RAL7016
   NP-CL-7016
   NP-SL-7016
   NP-CZ-7016
   NP-SZ-7016
   NP-T-7016
   NP-TF-7016
   POFÁK összesen
   CSAVAROK összesen
```

Ha be van pipálva → **két blokk**:

```
SZÍN: RAL7016

--- CIPZÁROS ---
   NP-CL-7016
   NP-CZ-7016
   NP-T-7016
   NP-TF-7016
   POFÁK (cipzáros)
   CSAVAROK (cipzáros)

--- SÍNES + BOWDENES ---
   NP-SL-7016
   NP-SZ-7016
   NP-T-7016
   NP-TF-7016
   POFÁK (sines+bowdenes)
   CSAVAROK (sines+bowdenes)
```

Ez **pontosan azt adja**, amit a festők kértek:

- szín szerint együtt kapják
- de a cipzáros külön kezelhető

**mind külön sorban jelennek meg**, így a szétválasztás triviális.

# ⭐ Mit kell hozzá tenni?

Csak egyetlen dolgot:

### → A Paint() függvényben, a kiírás előtt, csoportosítani kell az anyagokat típus szerint.

Például:

```cpp
QList<QUuid> cipzarosAnyagok;
QList<QUuid> sinesBowdenesAnyagok;

for (const QUuid& matId : materials) {
    const MaterialMaster* mat = MaterialRegistry::instance().findById(matId);
    QString barcode = mat ? mat->barcode : "";

    if (matchPrefix(barcode, "NP-CL") ||
        matchPrefix(barcode, "NP-CZ"))
        cipzarosAnyagok << matId;
    else
        sinesBowdenesAnyagok << matId;
}
```

És a checkbox szerint:

- vagy egyben írod ki
- vagy két blokkban
